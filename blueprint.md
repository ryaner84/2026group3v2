This document outlines the design, features, and development plan for the web application.

## Implemented Features

## Current Plan: Premium UI and Day/Night Mode

*   **Objective:** Redesign the UI to be more premium and add a day/night theme toggle.
*   **Steps:**
    1.  Create `style.css` for styling and `main.js` for interactivity.
    2.  Implement a day/night theme toggle button.
    3.  Define color palettes for both themes using CSS variables.
    4.  Add a subtle noise texture to the background for a premium feel.
    5.  Enhance visual depth with multi-layered drop shadows on elements.
    6.  Use modern typography with Google Fonts.
    7.  Ensure the layout is clean, balanced, and responsive.
